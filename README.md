# Promo Service

Данный проект был реализован как тестовое ```prod-like``` задание в компанию JetLend, основная цель которого продемонстрировать навыки разработки сервисов.

## Quick start

Для поднятия проекта необходимо склонировать репозиторий

```copy
https://github.com/Dezmound1/PromoService.git
```

После успешного клонирования, всё, что вам необходимо - применить команду в корне проекта

```bash
make build
```
а потом

```bash
make run
```
